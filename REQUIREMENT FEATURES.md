⚠️ Required Features (MVP)
Your project must include:
1. Input Form
-Fields: Item Name, Amount, Category (Food, Transport, Fun)
-Add a transaction to the list when submitted
-Validate that all fields are filled

2. Transaction List
-Scrollable list of all added items
-Show name, amount, and category
-Ability to delete items

3. Total Balance
-Display total balance at the top
-Updates automatically when items are added or deleted

4. Visual Chart
-Pie chart showing spending distribution by category
-Chart updates automatically when items change
-You can use Chart.js or another simple chart library
